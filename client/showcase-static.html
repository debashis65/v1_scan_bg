Barogrip UI Components Showcase

A showcase of the modernized UI components built with React, TypeScript, and Tailwind CSS

Authentication

Login Page

A secure login interface with role selection

src/pages/auth/LoginPage.tsx

Features:

  • Role-based login for doctors, patients, and administrators
  • Form validation with error feedback
  • "Remember me" functionality
  • Responsive two-column layout with information panel

Registration Page

Role-specific registration form

src/pages/auth/RegisterPage.tsx

Features:

  • Tab-based interface to switch between doctor and patient registration
  • Doctor-specific fields: specialty, license number, clinic, experience
  • Patient-specific fields: date of birth, gender, contact information
  • Password strength meter and validation
  • Terms and conditions agreement checkbox

Patient Detail Page
