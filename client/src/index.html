Barogrip UI Components

Below is a showcase of the modernized UI components for the Barogrip foot health platform. These components have been built using React, TypeScript, and Tailwind CSS.

Login Page

Location: src/pages/auth/LoginPage.tsx Features: - Role-based login (doctor, patient, admin) - Form validation - "Remember me" option - Password recovery link

Registration Page

Location: src/pages/auth/RegisterPage.tsx Features: - Role-specific fields for doctors and patients - Password strength indicator - Form validation - Terms and conditions agreement

Doctor Dashboard

Location: src/pages/doctor/DashboardPage.tsx Features: - Statistics overview (patients, scans, appointments) - Recent scans with status indicators - Today's appointments - Quick action buttons

Patient Dashboard

Location: src/pages/patient/DashboardPage.tsx Features: - Personal statistics and foot health score - Recent scans - Upcoming appointments - Treatment recommendations with completion tracking

Patient Detail Page

Location: src/pages/doctor/PatientDetailPage.tsx Features: - Patient personal and medical information - Tabbed interface for scans, appointments, and notes - Ability to add notes to patient record - Quick action buttons (book appointment, request scan)

Scan Detail Page

Location: src/pages/scan/ScanDetailPage.tsx Features: - Visualizations of foot scan data - AI diagnosis with confidence score - Treatment recommendations - Role-specific actions (download report, download 3D model)
